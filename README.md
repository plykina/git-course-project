# Project Chronicle
CLI-утилита для логирования действий.

**Сборка проекта**
`./gradlew build`

**Запуск тестов**
`./gradlew test`

**Использование**
`java -jar build/libs/git-course-project-1.0.0.jar add "Текст записи"`
`java -jar build/libs/git-course-project-1.0.0.jar list`
